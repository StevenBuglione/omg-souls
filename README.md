# OMG Souls

Public creative voice contracts for OMG automation.

A soul is style authority only. It may shape prose voice, tone, rhythm, humor,
viewpoint, and creative stance. It cannot override schemas, artifact naming,
safety gates, citation requirements, merge policy, publishing checks, or local
validation.

## Structure

- `souls.json` indexes available souls.
- `souls/<soul_id>/SOUL.md` contains the voice/style contract.
- `souls/<soul_id>/soul.json` contains metadata.
- `schemas/` contains JSON schemas for validation.

## Initial Soul

`lantern` is the first proof soul for illustrated wiki pages and future
book-engine experiments.
